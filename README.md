# Purrfect Typing

🐱‍🏍 **Purrfect Typing** is a sleek and responsive typing test application built with Laravel, Vue.js, Tailwind CSS, and Inertia.js. Enhance your typing speed and accuracy through interactive challenges designed to improve your keyboard proficiency.

## 📝 Description

Purrfect Typing offers users an engaging platform to practice and improve their typing skills. Featuring randomly generated word pools, real-time feedback on typing accuracy, and a clean, user-friendly interface, this application is perfect for both beginners and seasoned typists looking to sharpen their skills.

## 🚀 Features

- **Random Word Pools:** Generate a diverse set of words for varied typing exercises.
- **Real-Time Feedback:** Instant validation of typed characters with color-coded accuracy indicators.
- **Responsive Design:** Seamless experience across desktops, tablets, and mobile devices.
- **Interactive Cursor:** Dynamic cursor that follows your typing in real-time.
- **User-Friendly Interface:** Minimalistic and intuitive design for focused practice.
